## Naming Convention Examples

This recipe demonstrates the new flexible naming system:

- **Main steps**: `1.jpg`, `2.jpg`, `3.jpg`
- **Sub-steps**: `1.1.jpg`, `1.2.jpg` for detailed breakdowns
- **Legacy support**: `step-1.jpg` still works for backward compatibility

### File Structure
```
public/images/recipes/breakfast/avocado-toast-egg/steps/
├── 1.1.jpg    ← Cutting avocado (sub-step)
├── 1.2.jpg    ← Mashing avocado (sub-step)  
├── 2.jpg      ← Toasting bread (main step)
└── 3.jpg      ← Final assembly (main step)
```
